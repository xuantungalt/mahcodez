Building Python 2.7.10 on Raspberry Pi 2

  • software python2 rpi python
  • created, 01-13-2016
    updated, 02-01-2016
  • Blake VandeMerwe

  • Comments
Print

Install the latest version of Python 2.7 on your Raspberry Pi 2 in around ~20 minutes!

Depending on your Raspberry Pi operating system distribution you may not have access to the newest Python 2.7 release. Following this guide will help you get updated in no time. Feel free to leave a comment if you have any questions!

Let's Go

Make sure everything is up to date on your Raspberry Pi 2. libssl-dev is required before building Python for all the HTTPS/SSL related functionality to work correctly. tk-dev is required for Tk GUI development, and libsqlite3-dev will add Sqlite3, embedded SQL database, support.

Grab Pre-requisites

sudo apt-get update
sudo apt-get upgrade -y
sudo apt-get install build-essential libncursesw5-dev libgdbm-dev libc6-dev 
sudo apt-get install zlib1g-dev libsqlite3-dev tk-dev
sudo apt-get install libssl-dev openssl

Download and Make

$ cd ~
$ wget https://www.python.org/ftp/python/2.7.10/Python-2.7.10.tgz
$ tar -zxvf Python-2.7.10.tgz
$ cd Python-2.7.10
$ ./configure
$ make -j 4
$ sudo make install
  • wget took 6s on 30Mbps internet.
  • tar took 25s.
  • configure took 3m35s
  • make took 9m10s (with 4 concurrent jobs, ala make -j 4)
  • make install took 4m5s

The whole process took roughly 18 minutes to download, compile, and install Python 2.7.10 on a Raspberry Pi 2.

Test the Installation

$ python2

Python 2.7.10 (default, Jun  1 2015, 21:23:00)
[GCC 4.6.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>

Get Pip

$ cd ~
$ wget https://bootstrap.pypa.io/get-pip.py
$ sudo python2 get-pip.py

Test the Installation

$ pip --version
pip 7.0.1 from /usr/local/lib/python3.4/site-packages (python 3.4)
$ sudo pip install requests
$ python2
>>> import requests
>>>

Optionally, you can overwrite the included Python 3 version (3.2.3 on my distribution) by updating the symlink.

 sudo ln -sf /usr/local/bin/python2.7 /usr/bin/python

Congratulations! :)

>>> print 'Hello, World!'
