HOW TO POWER A RASPBERRY PI ZERO OVER ETHERNET
--

This guide will show you how to power a Raspberry Pi Zero over an ethernet cable. Since ethernet plugs only use half their pins, we can use the other half for simultaneous power.


It’s meant to be used with power over ethernet compatible networking hardware, so you can power it directly through the ethernet cables, without any batteries or USB chargers. You just plug it in, and you have both power and internet connectivity through a single cable.




PARTS


- Raspberry Pi Zero
- USB Ethernet adapter (make sure it is compatible with your Raspberry Pi)
- 12v-5v step down convertor (the input voltage depends on your networking hardware, but it should output 5v)
- 2x micro USB plugs
- Thin wires



TOOLS


- Soldering Iron
- Clippers
- Wire strippers
- Needle nose plyers



HOW-TO GUIDE


Step 1. Remove the casing from the USB ethernet adaptor


Step 2. Desolder the USB cable from the adaptor


Step 3. We need to make sure pins 4,5,7 and 8 on the ethernet socket are removed from the adaptor, so we can free them up for power. I tried removing the socket itself, but had no luck, so I ended up snipping the pins at the back and bending them out the way.


Step 4. We now need to create the Micro USB plugs for both the power, and the USB connection to the ethernet adaptor. To get them to work on the Pi, we need to short the ID and Ground pins like shown.


Step 5. Desolder the USB socket from the step down convertor.


Step 6. Now solder everything together and complete the circuit.


Step 7. Add some electrical tape to the parts so they don’t short on each other and put them inside a case. I made a simple one out of cardboard.


Finished. Now plug your Pi Zero into a compatible ethernet port  and watch it switch on! You can now log into the Pi remotely and use it as a headless computer.
